# Servlet Login Application

This is a simple Java Servlet Login system using HTML form.

### Features:
- Login form with username & password
- Servlet handles POST request
- Displays success or error message
- Can be deployed on Tomcat

### How to Run:
1. Import the project in Eclipse/IntelliJ.
2. Add Apache Tomcat server.
3. Run the project on server.
4. Open in browser:
   http://localhost:8080/Servlet-Login-App/
